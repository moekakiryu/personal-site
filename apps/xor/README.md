# XOR Encryption Tool

This directory contains the source code for a miniature encryption tool:

[https://xor.raymondlewandowski.dev](https://xor.raymondlewandowski.dev)

This tool can create an xor-encrypted string given a message and key.
Alternatively, if an encrypted message is included in the URL query params, the
tool will prompt for a password and display the decrypted message.

## Commands

This tool is a single static HTML file, no commands are required to develop it.

## Build

Since this is a static HTML file, there are no build steps. Please modify
`apps/xor/templates/xor/index.html` to make changes.
